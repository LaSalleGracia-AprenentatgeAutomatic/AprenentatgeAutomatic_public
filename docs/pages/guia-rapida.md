# Guia ràpida

## Objectius generals
- Comprendre els principis de l’aprenentatge automàtic (supervisat, no supervisat i profund).
- Implementar models bàsics (regressió, classificació i clustering) amb Python.
- Avaluar models amb mètriques adequades i interpretar resultats.
- Integrar matemàtiques essencials "just in time".

## Eines
Python, NumPy, Pandas, Matplotlib/Seaborn i scikit-learn.
